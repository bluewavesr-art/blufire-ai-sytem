## Project Delivery Standards
1. Every new build starts with a sitemap approved by the client
2. All SEO pages must have title tag, meta description, H1, 500+ words, schema markup
3. Mobile-first — every build tested on mobile before delivery
4. Page speed: Core Web Vitals must be Green before launch
5. SSL: All sites must have SSL — no exceptions
